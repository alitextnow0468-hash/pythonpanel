# پنل مدیریت نیروی انسانی - نسخه ماژولار

این نسخه به صورت ماژولار ساخته شده تا نگهداری و توسعه آینده آسان‌تر باشد.

## ساختار پروژه

```
pythonpanel/
├── main.py
├── resources/
├── tabs/
│   ├── home_tab.py
│   ├── dashboard_tab.py
│   ├── register_tab.py
│   ├── list_tab.py
│   ├── salary_tab.py
│   ├── leave_tab.py
├── utils/
│   ├── salary_calculator.py
│   ├── jalali_helper.py
│   ├── pdf_generator.py
├── build_exe.py
└── requirements.txt
```

## نحوه استفاده

فایل‌ها به صورت .txt در GitHub قرار دارند.
شما آن‌ها را در سیستم خود به .py تغییر نام دهید و اجرا کنید.
